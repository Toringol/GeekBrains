Сканеры сайтов можно использовать для анализа безопасности других веб-сайтов
Например, для анализа sql инъекций или анализа скрытых get параметров, command injection и тп
Можно использовать для парсинга страниц и в дальнейшем использовать этот контент у себя на сайте
Например, сайт кинотеатра может парсить сайт, где можно найти описание к фильму(например, кинопоиск) и
использовать этот контент у себя на сайте
